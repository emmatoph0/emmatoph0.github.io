📌 ¿Qué es la informática?

La informática es la ciencia que estudia el procesamiento automático de la información mediante computadoras. Es fundamental en la actualidad, ya que permite desarrollar tecnologías como aplicaciones, inteligencia artificial, redes y sistemas digitales que usamos todos los días.

🖥️ Hardware

El hardware es la parte física de una computadora. Incluye componentes como el procesador (CPU), la memoria RAM, el disco duro y los dispositivos de entrada y salida. Sin hardware, no existiría la base para ejecutar programas.

💾 Software

El software son los programas que permiten que la computadora funcione. Incluye sistemas operativos, aplicaciones, videojuegos y herramientas digitales. Es una de las áreas más importantes y con mayor crecimiento laboral.

👨‍💻 Lenguajes de Programación

Los lenguajes de programación permiten crear tecnología. Son fundamentales para desarrollar aplicaciones, páginas web y sistemas inteligentes. Aprenderlos abre muchas oportunidades laborales.

HTML

Lenguaje que estructura páginas web.

CSS

Permite diseñar y dar estilo visual.

Python

Lenguaje versátil usado en IA, datos y automatización.
